# Sauce Demo Automation 

## Overview
This project automates a basic **user journey** on [Sauce Demo](https://www.saucedemo.com/) using **Playwright** and **Page Object Model (POM)**.

**User Journey Tested:**
1. Login as standard_user
2. Open first product and add to cart
3. Go back to products page
4. Open second product and add to cart
5. View Cart and verify buttons:
   - Remove (per product)
   - Continue Shopping
   - Checkout

## Setup

1. Clone the repository:
git clone <your-repo-url>
npm install
npx playwright install

# Run all tests
npx playwright test

# Open HTML report after running tests
npx playwright show-report
