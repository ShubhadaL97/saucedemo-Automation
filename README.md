# saucedemo-test-automation
This repository contains an automated test framework for the SauceDemo web application and it covers user flow: Login → Open a product → Add to cart → Return to products page → Open another product → Add to cart → View Cart (validating buttons)

**User Journey Tested:**
1. Login as `standard_user`
2. Open first product and add to cart
3. Go back to products page
4. Open second product and add to cart
5. View Cart and verify buttons:
   - Remove (per product)
   - Continue Shopping
   - Checkout


## Project Structure
saucedemo-automation/
- tests/ # Test scripts
--- validateAddToCart.spec.js
- pages/ # Page Object Model files
--- loginPage.js
--- productsPage.js
--- cartPage.js
- screenshots/ # Screenshots captured during tests
- playwright-report/ # HTML report generated by Playwright
-package.json
-playwright.config.js
-README.md

## Setup

1. Clone the repository:
git clone https://github.com/ShubhadaL97/saucedemo-Automation.git
cd saucedemo-automation
npm install
npx playwright install

# Run all tests
npx playwright test

# Open HTML report after running tests
npx playwright show-report
